# Все о радиаторам Rifar Tubog

Готовая статическая вёрстка (TailwindCSS + DaisyUI через CDN).

## Файлы
- index.html — Главная
- models.html — Модели и характеристики
- installation.html — Монтаж и эксплуатация
- faq.html — FAQ
- robots.txt — открыт для основных поисковых ботов, закрыт для остальных
- sitemap.xml — карта сайта (замените домен `https://rifar-tubog-web.github.io` на свой)
- favicon.ico — положите в корень рядом с index.html
- 1.webp, 2.webp, ... — положите в корень. На страницах заложены до 19 изображений.

## Что настроить
1. Положите изображения `1.webp`… по списку в корень.
2. Положите `favicon.ico` в корень (ссылка уже прописана).
3. Замените домен в `sitemap.xml` и при необходимости в `<link rel="canonical">` в HTML-файлах.
4. Загрузите на хостинг (GitHub Pages, любой static hosting, Nginx).
